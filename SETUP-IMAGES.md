# How to Add Your Profile Image

## Option 1: Upload Image via GitHub (Recommended)

1. **Upload your image to your repository:**
   - Go to your portfolio repository on GitHub
   - Click "Add file" → "Upload files"
   - Select your profile photo (JPG, PNG, WebP recommended)
   - Name it: `profile.jpg` or similar
   - Commit the file

2. **Update the HTML:**
   - Open `index.html`
   - Find this line:
   ```html
   <img src="https://via.placeholder.com/400x400?text=Your+Photo+Here" alt="Jivan Parajuli" id="profileImage">
